HTML

HTML (HyperText Markup Language) is the foundation of web pages, providing the structure and content for websites.

CSS

CSS (Cascading Style Sheets) is the language that styles the visual appearance of web pages created with HTML, allowing you to control elements like layout, fonts, colors, and animations.

TAILWIND

A utility-first CSS framework that provides pre-designed utility classes to style your website directly within your HTML, offering a rapid development workflow without writing extensive custom CSS.

BOOTSTRAP

A free and open-source framework providing pre-built components and styles for responsive web design, simplifying and accelerating the development process.

JAVASCRIPT

A versatile and high-level programming language that adds interactivity and dynamic behavior to web pages, allowing for user interaction, animations, and complex web applications.

REACT JS

A popular JavaScript library for building user interfaces, known for its component-based architecture that promotes reusable code, efficient rendering, and a focus on user experience.

VUE JS

A progressive JavaScript framework for building user interfaces, offering a balance between simplicity and power with its approachable syntax, flexibility, and focus on reactivity.

NODE JS

An open-source JavaScript runtime environment that allows you to execute JavaScript code outside of a web browser, enabling the creation of server-side applications and scalable network tools.
